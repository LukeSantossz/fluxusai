# Instruções para Adicionar a Logo

Por favor, adicione o arquivo `logo.png` nesta pasta (`public/`).

O arquivo deve se chamar exatamente `logo.png` e será usado automaticamente pelo componente Logo.

## Localização do arquivo:
```
public/logo.png
```

## Tamanho recomendado:
- Para melhor qualidade: 512x256 pixels ou maior (proporção 2:1)
- Formato: PNG com fundo transparente (se possível)

O componente Logo já está configurado para usar `/logo.png` e se adaptará automaticamente aos diferentes tamanhos (sm, md, lg).
